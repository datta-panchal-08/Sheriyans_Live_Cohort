# 🌐 HTML Elements I Have Learned  

Mastering the basics of HTML is essential for web development. Here’s a list of HTML tags I have learned so far, along with their purposes:  

## 📌 Headings (`<h1>` - `<h6>`)  
Headings define the importance of text. `<h1>` is the most important, while `<h6>` is the least.  

```html
<h1>Main Heading</h1>
<h2>Subheading</h2>
<h6>Smallest Heading</h6>
```

## 📋 Lists (`<ol>`, `<ul>`, `<li>`)  
- **Ordered List (`<ol>`)**: Displays items in a numbered order.  
- **Unordered List (`<ul>`)**: Displays items with bullet points.  
- **List Item (`<li>`)**: Represents an item inside a list.  

```html
<ol>
  <li>First Item</li>
  <li>Second Item</li>
</ol>

<ul>
  <li>Bullet Point One</li>
  <li>Bullet Point Two</li>
</ul>
```

## ✍️ Formatting (`<p>`, `<b>`, `<i>`, `<sub>`, `<sup>`)  
- **Paragraph (`<p>`)**: Defines a block of text.  
- **Bold (`<b>`)**: Highlights important words.  
- **Italic (`<i>`)**: Emphasizes text with a slanted font.  
- **Subscript (`<sub>`)**: Lowers text, like chemical formulas.  
- **Superscript (`<sup>`)**: Raises text, often for exponents.  

```html
<p>This is a paragraph.</p>
<b>Bold Text</b>
<i>Italic Text</i>
H<sub>2</sub>O (Water)
10<sup>2</sup> = 100
```

## 📸 Adding Media (`<img>`)  
The `<img>` tag is used to display images.  

```html
<img src="image.jpg" alt="Description of image">
```

## 🔗 Hyperlinks (`<a>`)  
The `<a>` tag creates clickable links.  

```html
<a href="https://example.com">Visit Example</a>
```

## 🎨 Structure (`<br>`, `<hr>`)  
- **Line Break (`<br>`)**: Moves text to a new line.  
- **Horizontal Rule (`<hr>`)**: Creates a thematic break.  

```html
<p>First line.<br>Second line.</p>
<hr>
```
